## 1.ts 转换 js

安装 typescript

```bash
npm i -g typescript
```

tsc 命令 转换

```bash
tsc xxx.ts
```

通过 nodejs 运行 js 文件

```bash
node xxx.js
```

## 2.直接运行 ts 文件

安装 ts-node

```bash
npm i -g ts-node
```

通过 ts-node 运行 ts 文件

```bash
ts-node xxx.ts
```
